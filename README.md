# 🎓 EduNova AI Ultimate v10

EduNova AI is a modern AI-powered learning platform developed for students to improve their learning experience through Artificial Intelligence.

---

## 🚀 Features

- 🤖 AI Doubt Solver
- 📚 Smart Notes
- 📝 Mock Tests
- 🏆 Certificate Generator
- 📊 Progress Tracker
- 🥇 Leaderboard
- 👤 Student Profile
- ⚙️ Settings
- 📄 Previous Year Papers
- ⭐ Feedback System
- ❓ FAQ
- 🌙 Dark Mode
- 📱 Responsive Design
- ☁ Firebase Authentication
- 🤖 Google Gemini AI Integration
- 📥 Download Notes
- 🖨 Print Certificate
- 🔥 Daily Learning Streak
- 🏅 Achievement Badges
- 🎤 Voice Input
- 📜 Chat History
- 📡 Offline Support (PWA)

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Firestore
- Google Gemini API
- Progressive Web App (PWA)
